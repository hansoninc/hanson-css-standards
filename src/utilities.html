Direction

Set the direction of flex items in a flex container. In most cases you can omit the horizontal class here as the browser default is row. However, you may encounter situations where you needed to explicitly set this value (like responsive layouts).

Use .flex-row to set a horizontal direction (the browser default), or .flex-row-reverse to start the horizontal direction from the opposite side.

Flex Grow/Shrink Item

Often you need to set one column to shrink and one column to grow in a flexbox layout this can be achieved using flexbox grow, flexbox shrink, and flexbox auto

.flex-shrink will shrink the item down as small as it can go

.flex-auto will expand to the width of the content or until it runs into another column

.flex-grow will expand as far as it can go

BG Color

Utility creates a background color for each color in the $brand-colors array in scss > utilities > _utility-variables.scss. To add new colors simply add a new line to the Sass map for $brand-colors giving it a key and a color value. The class that is generated will be .bg-{map key} - (Ex. .bg-purple)

Pulls

Pull classes are standard to bootstrap, however they are not responsive by default. For example, you might put a .pull-right class on an object only to find out it doesnt work great at mobile. These pulls are responsive and allow you to just target specific breakpoints if needed. You also have the ability to use the pull-none class which doesnt exist in bootstrap to remove the float functionality at certain break point if needed

This pulls left at the medium breakpoint
This pulls right at the large breakpoint only
This sets float to "none" in case you need to overwrite something in rare cases

Positioning

Allows you to set the positioning class on an element, these are not built into bootstrap 3 but will be in bootstrap 4. Most like position-relative would be the most common use for creating containers for absolute positioned elements.

Text Alignment

This is also included in bootstrap 3 but is not responsive. This utility makes these styles responsive.

Text aligns right on lg and above
Text aligns left on md and above
Text aligns center on lg and above
